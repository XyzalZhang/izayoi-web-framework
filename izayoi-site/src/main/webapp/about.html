Dependencies

  • MVEL2 (http://mvel.codehaus.org/)

    一个基于 java 的强大 EL 解释器，也是当前被认为执行速度最快的 EL 解释器之一。

  • Xerces2 Java Parser (http://xerces.apache.org/xerces2-j/)

    一个著名的 XML SAX Parser 实现

  • dom4j (http://dom4j.sourceforge.net/)

    最常用的 XML DOM 工具包之一

  • PicoContainer (http://www.picocontainer.org/)

    一个高度嵌入式的 DI 微内核

  • Commons IO (http://commons.apache.org/io/)

    Apache 的输入/输出工具包

Relative Projects

  • WebWheel (http://www.webwheel.cn/)

    一个基于 java 的超轻量级 web 开发框架；正是在这个框架里，框架作者 kobe2000 首先提出了 Cortile 现在所发展的 html-friendly 的页面模板风格。

    WebWheel 是一个包含页面模板引擎、MVC、DI、数据库操作等在内的风格独特的全功能 web 框架，通过大量使用动态字节码生成，WebWheel 在保证代码精简与灵活性的同时追求运行效率的最大化。

  • Maven 2 (http://maven.apache.org/)

    izayoi web framework 所使用的 Java 项目管理工具，Maven 2 用一个格式统一的 xml 文件描述项目的名称、类型、版本、依赖等各项信息，从而避免因开发工具区别给编译源代码造成的困难。

    尝试使用 izayoi web framework 源代码最便捷的方式就是 checkout svn 然后使用 Maven 2 构建根目录下的 pom.xml。著名的 Java IDE 如 IDEA 及 Eclipse 都对 Maven 2 提供了很好的支持；或者 Maven 2 也提供了官方的命令行版本以供选择。

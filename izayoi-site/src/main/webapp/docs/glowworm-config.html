glowworm configuration

Glowworm 追求完全零配置的使用；但用户也可以在习惯上做一些微调。

要修改 Glowworm 的配置，放置包含附加内容的配置文件在 /WEB-INF/glowworm.properties (或者初始化 GlowwormLight filter 时通过参数 config-path 提供的其它路径）。

/WEB-INF/mock/shop/index.html.txt
